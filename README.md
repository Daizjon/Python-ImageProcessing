# Computer Vision and Image Processing (Python)

A structured collection of image processing and computer vision algorithms implemented in Python.

This repository contains both foundational techniques and applied projects covering spatial-domain filtering, frequency-domain processing, morphology, edge detection, and spectral restoration.



## Topics Covered

- Image I/O and basic operations
- Point operations and histogram processing
- Spatial filtering and smoothing
- Nonlinear rank filters
- Edge detection (Canny, Sobel, LoG)
- Hough transform (circle detection)
- Morphological operations and reconstruction
- Spectral filtering and resampling
- Frequency-domain restoration
- Butterworth notch filtering
- Applied mini-projects
